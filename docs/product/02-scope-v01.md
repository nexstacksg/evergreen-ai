# 02 — Scope v0.1

## In Scope
- Multi-source document intake (email, WhatsApp, folder)
- OCR + AI extraction + low-confidence highlighting
- Product matching with supplier-name learning map
- Verification UI (source vs extracted split view)
- ERP-format Excel export
- AI marketing copy + image enhancement + approval queue
- AI quotation suggestions (history + last price + price book)
- Customer portal ordering with:
  - Customer-specific price visibility (contract-only / all)
  - Consignee selection
  - Cost centre selection and budget checks
  - Sales order status lifecycle

## Out of Scope (Phase 1)
- Fully automated marketplace publish
- Fully automated ERP posting (direct write)
- Advanced warehouse routing/optimisation

## Dependencies
- Product catalogue master data
- Customer price book and validity windows
- Consignee and cost-centre data
- Historical sales/order data source
